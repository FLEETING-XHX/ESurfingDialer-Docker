# ESurfingDialer 文件夹说明

当前常用内容：

- Docker 软路由部署：`00_最终部署版本/ESurfingDialer-Itsuwarii-Docker-Fixed.zip`
- Windows 轻量客户端源码：`ESurfingDialer-Windows-Client/`
- Windows 轻量客户端安装包：`00_最终部署版本/Windows客户端_v1.00/ESurfingDialer-Lite-v1.00-Setup.exe`

## 目录用途

- `00_最终部署版本/`：软路由 Docker 最终可用版本，上传到 OpenWrt 时优先使用这里的 zip。
- `01_说明文档/`：问题分析、掉线恢复、部署说明等文档。
- `02_原始压缩包/`：最初拿到的原始 zip 和旧包备份。
- `03_旧版尝试/`：之前的中间尝试版本，仅作为归档。
- `04_Windows一键版/`：旧 Windows 一键版，和新客户端分开保存。
- `05_工作缓存归档/`：源码解压、下载、对比时用过的工作目录。
- `06_修改前备份/`：修改前自动备份的文件。
- `ESurfingDialer-Windows-Client/`：新开发的 Windows 轻量客户端工程。

## Windows 客户端版本规则

- 当前稳定测试版：`1.00`
- 后续修复 bug 或新增功能时，版本号递增为 `1.01`、`1.02`
- 发给同学测试时，只需要发送安装包，不需要发送源码目录

当前发给同学的文件：

```text
00_最终部署版本/Windows客户端_v1.00/ESurfingDialer-Lite-v1.00-Setup.exe
```

旧的未编号安装包已归档到：

```text
05_工作缓存归档/Windows客户端旧未编号安装包/
```
